## Local Development

> - Make sure you have a [`GNU Make`](https://www.gnu.org/software/make/), [`GNU Debugger (GDB)`](https://www.sourceware.org/gdb/), and [`GNU Compiler Collection (GCC)`](https://gcc.gnu.org/onlinedocs/).

```bash
# Build the app.
make build

# Unit tests.
make test

# Run the app with GDB.
make debug

# Run the app.
make run
```
